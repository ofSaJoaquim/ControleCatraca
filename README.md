Controle de saída para standalone de controle de acesso

Funciona com o protocolo wiegand, lê pelas pinos 2(D0) e 3(D1).
Lê a ID dos usuários de um controle de acesso, caso um usuário tente acionar o controle de acesso mais de uma vez em menos de 3 segundos vai disparar uma sirene e não vai acionar a saida.
Caso contrario a saída é ativa e a porta/catraca ativada._
